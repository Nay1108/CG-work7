## 二、实验原理

### 2.1 质点-弹簧模型

布料被离散化为 N×N 网格状的质点集合，质点之间通过三类弹簧相连：

- **结构弹簧（Structural）**：连接相邻网格点，保持布料的基本结构
- **剪切弹簧（Shear）**：连接对角线方向的点，防止布料剪切变形
- **弯曲弹簧（Bending）**：跨越两个网格点连接，增加布料的弯曲刚度

根据胡克定律，两质点间的弹力公式为：
$$f_{spring} = -k_s(|x_a - x_b| - l)\frac{x_a - x_b}{|x_a - x_b|}$$

其中 $k_s$ 为弹簧劲度系数，$l$ 为弹簧原长。同时引入阻尼力防止系统发散：
$$f_{damping} = -k_d v$$

### 2.2 数值积分方法

- **显式欧拉**：$x_{t+1} = x_t + v_t\Delta t$，$v_{t+1} = v_t + a_t\Delta t$
- **半隐式欧拉**：$v_{t+1} = v_t + a_t\Delta t$，$x_{t+1} = x_t + v_{t+1}\Delta t$
- **隐式欧拉**：使用定点迭代法近似求解 $v_{t+1} = v_t + a_{t+1}\Delta t$

### 2.3 球体碰撞处理

当布料质点与球体发生穿透时，沿法线方向将质点推至球体表面，并去除法向速度分量，保留切向速度。

### 3.2 三种积分方法对比结果

#### 3.2.1 显式欧拉（Explicit Euler）

**表现特征**：
- 在默认参数（k_s=10000, k_d=1.0）下，布料在极短时间内（约0.5秒）即发生严重数值爆炸
- 质点速度呈指数级增长，布料迅速飞散破碎
- 即使设置了速度钳制（max_velocity=50.0），仍无法阻止系统发散

**稳定性分析**：
- 显式欧拉是条件稳定的，稳定条件为 $\Delta t < 2\sqrt{m/k}$
- 当弹簧刚度较大时，需要极小的 $\Delta t$ 才能保持稳定
- 实验中 $\Delta t = 5\times10^{-4}$ 对于 k_s=10000 仍然过大
- 只有在刚度较低（k_s < 1000）且阻尼较大（k_d > 5）时才能勉强保持稳定

**适用场景**：仅适用于低刚度、高阻尼的简单系统，不适合实际布料模拟

#### 3.2.2 半隐式欧拉（Semi-Implicit Euler）

**表现特征**：
- 在默认参数下表现稳定，布料自然下垂形成褶皱
- 运动轨迹平滑，具有良好的物理真实性
- 随着时间推移，布料逐渐稳定在平衡位置

**稳定性分析**：
- 半隐式欧拉是**辛积分器**，具有能量守恒特性
- 即使在大时间步长下也能保持系统稳定性
- 虽然速度更新使用显式方式，但由于位置更新使用新速度，系统整体稳定性大幅提升

**适用场景**：最适合实时布料模拟，在稳定性和效率之间取得良好平衡

#### 3.2.3 隐式欧拉（Implicit Euler）

**表现特征**：
- 系统极其稳定，即使使用较大的时间步长也不会发散
- 布料运动表现出明显的阻尼特性，收敛速度快
- 相比于半隐式欧拉，运动略显"粘滞"，缺乏弹性感

**稳定性分析**：
- 隐式欧拉是**无条件稳定**的，任何时间步长下都不会发散
- 使用定点迭代法（3次迭代）近似求解未来状态
- 数值阻尼效应明显，会快速耗散系统能量

**适用场景**：适合需要极致稳定性或大时间步长的场景，如离线渲染
#### 三种积分方法的不同效果


### 3.3 弹簧类型对布料形态的影响

**结构弹簧（Structural）**：
- 提供布料的基础骨架，决定网格的拉伸刚度
- 增大 k_s 使布料更加刚硬，形变减小
- 减小 k_s 使布料柔软，悬垂感更强

**剪切弹簧（Shear）**：
- 防止布料在对角线方向上的剪切变形
- 增大 k_shear 使布料在剪力作用下保持形状
- k_shear=0 时布料会呈现明显的"菱形"变形

**弯曲弹簧（Bending）**：
- 增加布料的抗弯刚度，使布料更加"挺括"
- 增大 k_bend 使布料不易产生褶皱
- k_bend=0 时布料折叠更自然，褶皱更丰富

**实验观察**：
- 纯结构弹簧（k_shear=k_bend=0）：布料柔软但缺乏结构感，容易产生不自然的拉伸
- 添加剪切弹簧：布料抵抗剪切变形，保持网格完整性
- 添加弯曲弹簧：布料具有抗弯能力，悬垂效果更加自然

### 3.4 阻尼系数的影响

**低阻尼（k_d < 0.5）**：
- 系统振荡时间长，布料持续摆动
- 显式欧拉在此参数下极易发散

**中等阻尼（k_d = 1.0）**：
- 系统在1-2秒内收敛至平衡位置
- 运动自然，具有真实的能量耗散效果

**高阻尼（k_d > 5.0）**：
- 系统快速收敛，几乎无振荡
- 布料运动显得"沉重"，缺乏弹性感

### 3.5 球体碰撞效果分析

**碰撞处理机制**：
1. **位置修正**：穿透球体的质点沿法线方向被推至球面
2. **速度修正**：去除法向速度分量，仅保留切向速度
3. **排斥力辅助**：在力计算阶段额外施加排斥力，使布料自然滑过球面

**交互效果**：
- 布料与球体接触时产生自然的包裹效果
- 球体位置可实时拖动，布料动态响应
- 关闭碰撞后布料自由穿过球体，凸显碰撞处理的重要性

## 四、结论

本实验成功实现了基于质点-弹簧系统的布料模拟，通过对比三种数值积分方法，得出了以下结论：

1. **显式欧拉**在刚度较大的系统中极易发散，不适合实际应用
2. **半隐式欧拉**在稳定性和效率间取得平衡，是实时模拟的最佳选择
3. **隐式欧拉**无条件稳定但具有数值阻尼，适合需要极致稳定性的场景
4. 增加剪切和弯曲弹簧能显著提升布料模拟的真实感
5. 球体碰撞处理有效增强了场景的交互性
